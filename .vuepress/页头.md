# 自定义头部参数
- title: 标题
- date：创建日期
- sticky：文章置顶，按照 1, 2, 3, ... 来降序排序
- lastUpdated：更新时间 true/false 
- NoPageCard: 是否使用卡片
- isLabelImage: 详情页是否显示图片，默认false, true/false 
- description: { 描述
  1. name： 描述内容
  2. detailIcon: 描述图标
  3.  detailIconSize: 图标大小
  4. detailShow: 详情页是否显示描述，默认true, true/false 
- }
- editLink：是否可编辑
- tags: 标签
- categories：分类
- permalink：页面链接
- pageClass: 背景类名
- prev：{ 上一页
  1. text: 标题
  2. link: 链接
- }
- next{ 下一页
  1. text: 标题
  2. link: 链接
- }
